Astar Block Explorer

Inspect every block, transaction, extrinsic, smart contract, and XCM transfer on Astar Network. Two complementary explorers give you full visibility: Subscan for Substrate-native data including extrinsics and events, and Blockscout for EVM transaction details and contract verification.

Overview

Explorer Overview

Two block explorers provide complementary views of the same on-chain data — choose based on what you need.

🔎

Subscan

astar.subscan.io is Astar's primary Substrate explorer. It exposes native Polkadot/Substrate data structures including extrinsics, events, runtime calls, dApp Staking eras, XCM messages, and on-chain governance. Essential for developers working with @polkadot/api or monitoring parachain activity.

Substrate Native XCM Tracking dApp Staking
🔍

Blockscout

blockscout.com/astar provides an Etherscan-compatible EVM explorer. Verify smart contracts, inspect internal transactions, decode ABI calls, view token transfers (ERC-20 / ERC-721), and monitor contract events. Supports Sourcify-based contract verification via the standard Hardhat/Foundry plugins.

EVM Native Contract Verify Token Transfers
Features

What You Can Do

Block explorers on Astar offer functionality across both EVM and Substrate layers.

🔁

Search Transactions

Look up any transaction by hash. View status (success/reverted), gas used, input data, emitted events, and internal calls. Works for both EVM and native Substrate extrinsics.

📋

Inspect Contracts

View verified contract source code, ABI, and read/write functions directly from Blockscout. Test contract calls without a frontend using the built-in read/write UI.

📦

View Extrinsics

On Subscan, inspect every Substrate extrinsic: its module, call, parameters, and resulting events. Useful for debugging dApp Staking, XCM transfers, and governance votes.

🌐

Track XCM Transfers

Subscan provides dedicated XCM message tracking — follow cross-chain asset transfers across Polkadot parachains, see origin, destination, and delivery status end-to-end.

Guide

Using the Explorer

Step-by-step guides for the most common explorer tasks on Astar.

Verify a Smart Contract on Blockscout

  1. Deploy your contract

    Deploy to Astar Mainnet or Shibuya Testnet. Note the contract address from your deployment output.

  2. Navigate to the contract on Blockscout

    Go to blockscout.com/astar and search for your contract address. Click the "Code" tab.

  3. Select verification method

    Choose "Flattened source code" or "Standard JSON Input". Match the Solidity version and optimization settings used during compilation.

  4. Submit and confirm

    Upload your source or JSON and submit. Blockscout will verify bytecode matches. Verified contracts show a green checkmark and expose read/write UI.

Track an XCM Transfer on Subscan

  1. Initiate the XCM transfer

    Send an XCM transfer from the Astar Portal or via polkadotXcm.limitedReserveTransferAssets extrinsic. Copy the extrinsic hash.

  2. Find the extrinsic on Subscan

    Go to astar.subscan.io, search the extrinsic hash. Click "XCM Messages" to see the associated cross-chain message.

  3. Follow the message

    Subscan shows the message hash, origin chain, destination chain, and delivery status. Click through to the destination chain's Subscan to confirm receipt.
