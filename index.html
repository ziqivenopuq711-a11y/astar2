Overview

What is Astar?

Astar Network is a Polkadot parachain that provides a multichain smart contract platform supporting multiple virtual machines and cross-chain interoperability.

⬡

Polkadot Parachain

Astar operates as a parachain on the Polkadot relay chain, inheriting its pooled security via Nominated Proof-of-Stake (NPoS) and benefiting from Polkadot's shared validator set — no separate token bonding required for chain security.

⚙️

Dual VM: EVM + WASM

Deploy Solidity smart contracts via the EVM (powered by Frontier) or use ink! contracts in WebAssembly. Both environments co-exist on the same chain, allowing developers to choose their preferred stack while users share unified state and liquidity.

💎

dApp Staking (Build2Earn)

Astar's unique dApp Staking mechanism lets token holders stake ASTR directly on registered dApps. Developers earn block rewards proportional to staked tokens — creating a sustainable income stream for builders without relying solely on transaction fees.

🔗

XCM Cross-Chain

Cross-Consensus Messaging (XCM) enables trustless asset transfers and contract calls across every Polkadot parachain. Move DOT, USDT, and other parachain assets into Astar dApps natively, unlocking composability across the entire Polkadot ecosystem.

Native cross-consensus messaging. Transfer assets to and from any Polkadot parachain with cryptographic security guarantees.

Quick Start

Start Building

From zero to deployed smart contract on Astar in five steps.

  1. Add Astar to MetaMask

    Open MetaMask, go to Networks → Add Network. Enter RPC https://evm.astar.network, Chain ID 592, symbol ASTR. For testing, add Shibuya with RPC https://evm.shibuya.astar.network, Chain ID 81, symbol SBY.

  2. Get Test Tokens from Astar Portal

    Visit portal.astar.network, connect your wallet, and request free SBY tokens from the Shibuya faucet. Tokens arrive within seconds — no Twitter verification required.

  3. Configure Hardhat or Foundry

    Add Astar networks to your config. For Hardhat: astar: { url: "https://evm.astar.network", chainId: 592 }. For Foundry: --rpc-url https://evm.astar.network --chain-id 592. Shibuya uses chainId 81.

  4. Deploy and Verify

    Deploy your contract using npx hardhat run scripts/deploy.js --network shibuya. Verify on Blockscout using the standard verification API compatible with Etherscan plugins.

  5. Register for dApp Staking

    Once deployed on mainnet, register your dApp at portal.astar.network under dApp Staking. Token holders can then stake ASTR on your project, earning you block rewards proportional to staked amounts.
